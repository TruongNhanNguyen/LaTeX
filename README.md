### Quick **LaTeX** overview

#### What is **LaTeX**

> **LaTeX â€“ A document preparation system**

**LaTeX** is a high-quality typesetting system; it includes features designed for the production of technical and scientific documentation. **LaTeX** is the de facto standard for the communication and publication of scientific documents. **LaTeX** is available as free software.

You don't have to pay for using **LaTeX**, i.e., there are no license fees, etc. But you are, of course, invited to support the maintenance and development efforts through a donation to the TeX Users Group (choose **LaTeX** Project contribution) if you are satisfied with **LaTeX**.

You can also sponsor the work of **LaTeX** team members through the **GitHub** sponsor program at the moment for **Frank**, **David** and **Joseph**. Your contribution will be matched by **GitHub** in the first year and goes **100%** to the developers.

For more information, you can visit this website [![LaTeX](https://img.shields.io/badge/-LaTeX_project.org-000000?style=flat&logo=latex&logoColor=ffffff&labelColor=563D7C)](https://www.latex-project.org/)

## My Installation

I use **Visual Studio Code** to write **LaTeX** documents. So i will share you my **Visual Studio Code** set up for **LaTeX**:

1. Install **TeXLive** (latest. version is **TeXLive** 2021)

2. Install **[![Visual Studio Code](https://img.shields.io/badge/-LaTeX_Workshop-000000?style=flat&logo=visual-studio-code&labelColor=007ACC)](https://marketplace.visualstudio.com/items?itemName=James-Yu.latex-workshop)** for **Visual Studio Code** from the **VSCode Marketplace** or directly install from **VSCode extensions** by press `Ctrl + Shift + X`, type **LaTeX workshop**, then just click `install`. Or use `Ctrl + P` and type `ext install Jame-Yu.latex-workshop`.

3. Then, open the editor, create new file and save with name `<name>.tex` and begin your typesetting.
